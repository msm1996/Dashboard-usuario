<h1>📊 Dashboard de Usuários</h1>
<h2>Projeto desenvolvido com React para exibição de usuários, com classificação por níveis (Iniciante, Intermediário, Avançado), gráficos e navegação entre páginas.</h2>
<br>
<h1>🚀 Funcionalidades</h1>

<h2>📥 Consumo de API
🔍 Busca de usuários
<br>
📊 Gráfico com níveis dos usuários
<br>
👤 Página de detalhes do usuário
<br>
🔄 Navegação entre páginas
<br>
🎨 Interface responsiva</h2>

<h1>🛠️ Tecnologias utilizadas</h1>
<h2>⚛️ React.js
<br>
🟨 JavaScript
<br>
🔗 Axios
<br>
🎨 Styled-components
<br>
📊 Recharts
<br>
🌐 React Router
<br>
🔌 API REST</h2>
<br>
<h1>🎯 Objetivo</h1>
<h2>Este projeto foi desenvolvido com o objetivo de praticar:
<br>
- Consumo de APIs
<br>
- Gerenciamento de estado
<br>
- Componentização
<br>
- Navegação entre páginas
<br>
- Exibição de dados dinâmicos</h2>
